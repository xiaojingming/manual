# 命令

> CSC 中可用命令的完整参考，包括内置命令和捆绑技能。

命令用于在会话内控制 CSC。它们提供了一种快速切换模型、管理权限、清除上下文、运行工作流等的方式。

输入 `/` 可查看所有可用命令，或输入 `/` 后跟字母进行筛选。

下表列出了 CSC 中包含的所有命令。标记为 **[Skill]** 的条目是捆绑技能。它们使用与你自行编写的技能相同的机制：将提示交给 Claude，Claude 也会在相关时自动调用。其他所有条目都是内置命令，其行为已编码到 CLI 中。要添加你自己的命令，请参阅技能。

并非每个命令都会对所有用户显示。可用性取决于你的平台、计划和环境。例如，`/desktop` 仅在 macOS 和 Windows 上显示，`/upgrade` 仅在 Pro 和 Max 计划中显示。

## MCP 提示

MCP 服务器可以暴露作为命令出现的提示。这些使用 `/mcp__<server>__<prompt>` 格式，并从连接的服务器动态发现。有关详细信息，请参阅 MCP 提示。
